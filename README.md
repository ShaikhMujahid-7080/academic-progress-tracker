### 🌟 Vision
**Academic Progress Tracker** isn't just a form; it's a sophisticated companion for your educational journey. Designed with a modern **Glassmorphism** aesthetic, it provides a seamless and visually stunning experience for students to monitor their growth.

---

### 🛡️ Core Features

#### 👨‍🎓 **For Students**
- **📚 Smart Assessment Tracking**: Real-time monitoring of CA 1-4 and Mid-Sem marks.
- **🚥 Dynamic Status Engine**: Blocks turn <kbd>Green</kbd> on completion and <kbd>Red</kbd> when deadlines expire.
- **📅 Intelligent Calendar**: Synchronized with IST (Asia/Kolkata) with automated event tracking.
- **🧪 Lab Experiment Monitor**: Visual tracking of practical completion.
- **📝 Academic Journal**: Semester-wise personal notes for quick reference.

#### ⚡ **For Administrators**
- **📌 Global Notice Board**: Post and pin critical assessments across all student dashboards.
- **👥 Branch Management**: Specialized curriculum views for **IT**, **AIML**, **DS**, and more.
- **🛠️ Curriculum Orchestrator**: Collapsible suite for managing subjects and assessment counts.

---

### 🎨 Design Philosophy
Inspired by **modern premium web design**, the app features:
- **Glassmorphism**: Elegant transparency and blurred backgrounds.
- **Micro-animations**: Subtle transitions that breathe life into the UI.
- **Responsive Harmony**: A flawless experience across Mobile, Tablet, and Desktop.

### 🚀 Quick Start

1️⃣ **Clone the Masterpiece**
```bash
git clone https://github.com/ShaikhMujahid-7080/academic-progress-tracker.git
```

2️⃣ **Assemble Dependencies**
```bash
npm install
```

3️⃣ **Initialize Environment**
Create a `.env` file with your Firebase credentials.

4️⃣ **Ignite Engine**
```bash
npm run dev
```
